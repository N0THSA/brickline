How to connect:

Make sure you have Novetus URI installed.

If you're on Linux you can try this NovetusURI installer.

  1. To install it, open up NovetusBootstrapper as administrator.

  2. Then, click on "INSTALL URI" at the bottom right inside of "other options".

  3. Once the pop up window opens, click "Install URI and UserAgent". Then you're done, you can now join any server you want just by clicking on the "Launch Novetus" button under the server details


How to host:

  1. Go to this link and download: addon.zip

  2. Extract the contents of addon.zip inside of Novetus\Addons\ and you're done

  3. Change the server name by editing the "ServerName" variable at the top of "Brickline.lua".


    4. Note: If you're using other custom addon scripts, do not replace AddonLoader.lua. Instead, open the file and manually add the string "Brickline" to the Addons table.

How to configure your server for playit.gg

    Playit.gg is required due to a potential exploit. Here are instructions to configure the addon for Playit.gg:

  1. Open up Brickline.lua

  2. Change the PLAYIT_ADDR to your respective Playit domain "e.g. mygame.playit.gg" and the PLAYIT_PORT to your respective port number you got assigned from Playit
